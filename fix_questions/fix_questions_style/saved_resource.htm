                Note: This comment is in reference to an older version of the editing process.

                Fixing these is fun :-)
                Tip: if you use Notepad++, you can make the job easier by first copying everything up to the start of the questions into a new file, then right-clicking the file tab and choosing "move to other view", then moving the cursor to a blank line, going to Macro > Record Macro, hitting enter once, then Macro > Stop recording, then going to right before the questions, subtracting the line number from the question number, and then going to Macro > Run a macro multiple times, and running the currently recorded macro the number of times equal to the result that you got.
